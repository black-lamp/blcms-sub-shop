Installation
------------
Run command
```
composer require black-lamp/blcms-sub-shop
```
or add
```json
"black-lamp/blcms-sub-shop: "dev-master"
```
to the require section of your composer.json.

#### Applying migrations
```
php yii migrate --migrationPath=@vendor/black-lamp/blcms-sub-shop/src/migrations
```
